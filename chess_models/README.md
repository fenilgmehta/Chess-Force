# Model weight file naming convention

FileName: **v`ModelVersion`-mg`ModelGenerator`-be`BoardEncoder`-sn`ScoreNormalizer`-ep`Epochs`-`weight|model`.`FileExtension`**

Description: 
- prefix
- ModelGenerator: 03d
- BoardEncoder: 05d
- ScoreNormalizer: 03d
- Epochs: 05d
- type: weights|model
- ModeVersion: 03d
- FileExtension: h5|hdf5
